# Access Management System

An automated system designed to audit and manage user permissions within business environments.

## Features
- **Auditing:** Analyzes access logs to identify unusual activity.
- **Reporting:** Generates clear security reports for informed decision-making.

## How to run it
1. Ensure you have Python installed.
2. Run the script using the following command: `python access_report.py`

## Project Status
Currently under active development as part of a security automation initiative.
